# Copyright Registration Guide - Legacy Translations Portal

## Registration with U.S. Copyright Office

### Step 1: Go to copyright.gov
Visit: https://www.copyright.gov/registration/

### Step 2: Create an Account
- Click "Register a Copyright"
- Create an account at https://eco.copyright.gov/

### Step 3: Fill Out the Application

**Type of Work:** Literary Work (software is classified as literary work)

**Title of Work:**
```
Legacy Translations Portal - Translation Management System
```

**Year of Completion:** 2024-2025

**Date of First Publication:** [Date you first made it available]

**Author Information:**
```
Name: [Your Full Legal Name]
Citizenship: [Your Citizenship]
Domicile: United States
Year of Birth: [Your Birth Year]
Author's Contribution: Entire software application including source code, user interface, and database design
Work Made for Hire: No (unless you created it as employee of a company)
```

**Claimant Information:**
```
Name: [Your Name or Company Name]
Address: [Your Address]
```

### Step 4: Upload Deposit Material

For software, you need to submit a "deposit" of your code. Options:

**Option A - Standard Deposit (Recommended for proprietary software):**
- First 25 pages of source code
- Last 25 pages of source code
- Remove trade secrets/sensitive parts if needed

**Option B - Special Relief (If code contains trade secrets):**
- Submit with portions blocked out
- Requires written request explaining why

### Step 5: Pay the Fee
- Standard online registration: $45 (single author)
- Standard online registration: $65 (multiple authors or work for hire)

## Important Notes

1. **Registration vs. Protection**: Your code is automatically protected by copyright
   when created, but registration provides legal benefits for enforcement.

2. **Timeline**: Processing takes 3-6 months typically.

3. **Certificate**: You'll receive a certificate of registration which serves as
   prima facie evidence of ownership.

4. **Updates**: Major updates can be registered as derivative works.
