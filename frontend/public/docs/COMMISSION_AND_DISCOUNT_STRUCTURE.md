# Legacy Translations - Estrutura de Comissões e Descontos

## Parte 1: Compensação do Vendedor

### Opção A: 100% Comissão (Sem Salário Fixo)
**Ideal para:** Vendedores experientes, freelancers, representantes comerciais

| Tipo | Valor | Quando Paga |
|------|-------|-------------|
| **Bônus por Partner Fechado** | $100 - $300 | Após 1º pedido do partner |
| **Comissão Recorrente** | 8-10% da receita | Mensal, enquanto partner ativo |
| **Bônus de Performance** | $500 - $2,000 | Ao atingir metas |

**Exemplo de Ganhos (100% comissão):**
```
Mês 1: Fecha 5 partners
- Bônus inicial: 5 x $150 = $750
- Receita dos partners: $1,500
- Comissão 10%: $150
- Total Mês 1: $900

Mês 6: 25 partners ativos
- Receita mensal dos partners: $12,000
- Comissão 10%: $1,200
- Novos partners (3): $450
- Total Mês 6: $1,650

Mês 12: 50 partners ativos
- Receita mensal: $30,000
- Comissão 10%: $3,000/mês
```

### Forma de Pagamento ao Vendedor

**Opções:**

| Método | Frequência | Observação |
|--------|------------|------------|
| **Zelle** | Semanal ou Quinzenal | Mais rápido, sem taxas |
| **ACH/Transferência** | Quinzenal ou Mensal | Tradicional |
| **PayPal** | Semanal | Para freelancers |
| **Contractor Payment (1099)** | Mensal | Com relatório fiscal |

**Calendário de Pagamento Sugerido:**
- **Bônus por partner:** Toda sexta-feira (partners fechados na semana)
- **Comissão recorrente:** Dia 15 de cada mês (referente ao mês anterior)
- **Bônus de performance:** Final de cada trimestre

---

### Contrato com Vendedor

**Modelo: Independent Contractor (1099)**

Cláusulas importantes:
1. Não é funcionário (sem benefícios, impostos retidos)
2. Comissão paga apenas para partners que permanecem ativos
3. Se partner cancelar em 30 dias, bônus pode ser estornado
4. Exclusividade opcional (maior comissão se exclusivo)
5. Prazo de 90 dias para "lock" do partner (depois qualquer vendedor pode atender)

### Sistema de Tiers Automático

O sistema pode ajustar automaticamente o tier baseado no histórico:

```
Lógica do Sistema:
1. Verificar média de páginas dos últimos 3 meses
2. Aplicar tier correspondente
3. Upgrade: imediato quando atinge volume
4. Downgrade: apenas após 2 meses abaixo do tier
```

**Exemplo:**
- Janeiro: 30 páginas → Tier 18%
- Fevereiro: 60 páginas → Tier 22% (upgrade imediato)
- Março: 45 páginas → Mantém 22% (proteção)
- Abril: 40 páginas → Mantém 22% (proteção)
- Maio: 35 páginas → Volta para 18% (2 meses abaixo)

### Cálculo Automático de Preço:

```javascript
function calculatePartnerPrice(basePrice, partner) {
  let discount = getVolumeDiscount(partner.monthly_average);

  // Adicionar descontos extras
  if (partner.annual_contract) discount += 0.03;
  if (partner.prepaid_7_days) discount += 0.02;

  // Limite máximo de desconto
  discount = Math.min(discount, 0.40);

  return basePrice * (1 - discount);
}
```
